About the Demo

What is Shown in the Demo

The demo displays a simulated intersection with working traffic lights. Traffic is generated on the roads, and the system recalculates and updates the traffic light phase every 60 seconds. This demonstrates how the model responds to changes in traffic load.

Problem & Solution Connection

The problem is that most urban traffic lights operate on fixed schedules and don't account for real road conditions, leading to traffic jams.

Our solution is an intelligent traffic light management system that analyzes traffic parameters (vehicle count, queue length, flow intensity) and automatically suggests optimal timing adjustments.

Tech Stack & AI Solutions

Technologies: Java, Python, Docker, AWS EC2, JavaScript, React

AI Implementation: Built using Scikit-learn with Random Forest model
